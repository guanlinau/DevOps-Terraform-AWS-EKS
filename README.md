### Terraform & AWS EKS

### Technologies used:

Terraform, AWS EKS, Docker, Linux, Git

### Project Description:

Automate provisioning EKS cluster with Terraform

### Usage Instructions:

```
terraform init
```

```
terraform plan
```

###### Create VPC and EKS cluster

```
terraform apply
```

###### Destroy vpc and EKS cluster

```
terraform destroy
```
